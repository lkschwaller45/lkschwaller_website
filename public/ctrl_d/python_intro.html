Writing the first script

Once you've gotten this far pycharm should have given you an initial python file called main. Go ahead and delete the contents of this file, as we'll rewrite everything we need! Today we want a really simple program, one that might not be really useful in our daily lives, but will cover most basics of python. As such, we're going to rely on good ol' high school algebra and create a program that when given an array of x values will output an array of y values based on an equation we define.

Mathematics

Let's back up for a brief second and explain the math we want to achieve here. An equation is an expression that tells us how two (or more) things relate. You'll probably know the classic equation

Y = M * X + B
This equation is the equation of any line, and is controlled by the x you input to get a y value. In this equation, M is the slope (or how quickly your line rises) and B is the y intercept (where the line crosses the y-axis of a graph. Typically when looking at an equation you think of entering one x value to get one y value, but in our case we want to feed this equation an array (or commonly called a list) of x values, and have it give us an array of y values. By doing this, we can get enough points that if we wanted to, we could graph the line itself! The final thing to understand, is that sometimes these equations are called functions if you look at them in broader terms.

Coding the Equation

My goal here is to keep this extremely simple so that this could be your first program, so let's write this equation as just a simple function in a program. The correct name for a function in a program is a "Method". Methods allow us to feed input into them, and return outputs. We can define a method by using the key word def (as in definition). Let's refer to this first method as "Line_Calculator" 


									def Line_Calculator(x_values):
In this method defintion, we've used the keyword def, alongside a title that we chose. Additionally, by using (x_values) we've instructed our method to expect that we will be giving it an input of a variable known as a x_values. Now we know that we expect our x_values variable to be an array of numbers that we wish to input, but at this time our program actually has no idea what x_values is, to it is only a name.

Now let's keep this first draft really simple, and say we only care about one slope, and one y-intercept. As such, let's plot a rather boring line, with a slope of 1, and an intercept at 0. Now if we think of the math behind this line, we can already know that this means we will get out a y value of exactly what we put in for our x value, as 1 times any number is 1. Let's write this out though! 

									
										def Line_Calculator(x_values):
											y_values = 1 * x_values
											return y_values
